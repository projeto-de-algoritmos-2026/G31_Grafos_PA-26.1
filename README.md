## Comandos

## Build

Rode o comando abaixo para buildar o projeto

```
ng build
```

## Rodar o projeto

Rode o comando abaixo para rodar o projeto

```
`ng serve` 
```

Siga para [http://localhost:4200/](http://localhost:4200/) para ver a aplicação rodando

## Gerar componentes

Rode o comando abaixo para gerar um novo componente

```
ng generate component component-name
```

Você também pode usar `ng generate directive|pipe|service|class|guard|interface|enum|module`.
