## Sobre 
Nesse projeto, pegamos um arquivo com coordenadas geográficas pertencentes às ruas do Gama e limitamos às ruas do bairro do Setor Industrial. Em seguida, geramos um grafo com essas coordenadas e usando BFS fizemos uma aplicação onde o usuário selecionada dois pontos no mapa e verifica se existe um caminho que conecta esses pontos só com as coordenadas que usamos.

## Instalação 
Linguagem: Typescript<br>
Framework: Angular<br>

## Dependências

Este projeto foi desenvolvido com as seguintes tecnologias:
- **Node.js** (recomendado `v18.x` ou versão compatível)
- **Angular CLI** versão `15.x`

## Instalando dependências

Antes de executar o projeto, instale as dependências do repositório (como o Angular, Leaflet, etc.) rodando o comando abaixo na raiz do projeto:

```bash
npm install
```

## Build

Rode o comando abaixo para buildar o projeto

```
ng build
```

## Rodar o projeto

Rode o comando abaixo para rodar o projeto

```
`ng serve` 
```

Siga para [http://localhost:4200/](http://localhost:4200/) para ver a aplicação rodando
